# 목재 최적 판매 계산기

`index.html` 하나만 GitHub 저장소에 올려도 동작합니다.

## 사용 방법

1. GitHub 저장소를 만듭니다.
2. `index.html` 파일을 저장소 루트에 올립니다.
3. GitHub Pages를 켭니다.
4. 페이지에 접속해서 보유 수량과 100개당 가격을 입력합니다.

## 계산 기준

- 판매는 모두 100개 단위
- 교환은 문제에서 준 정수 단위만 허용
- 교환 없이 바로 판매한 금액과 비교해서 추가 이득 표시
- 거래소 수수료는 미반영
- 초기 벌목의 가루는 선택 입력이며 기본값은 0
